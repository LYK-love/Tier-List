# Tier List Maker

[中文](README.md) | [English](README.en.md)

![Demo](assets/example.png)

A pure-frontend drag-and-drop tier list maker: create items, drag them into tiers, freely add/remove/rename tiers, and export the result as a PNG.  
Just open `index.html` (double-click). State is persisted in the browser via localStorage.
