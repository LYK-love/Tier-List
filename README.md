# Hot-to-Lame Tier List Maker

[English](README.md) | [中文](README.cn.md)

![Demo](assets/example.png)

A pure-frontend drag-and-drop tier list maker: create items, drag into tiers, add/remove/rename tiers, and export to PNG.  
Just open `index.html` (double-click) — state is persisted in the browser via localStorage.
