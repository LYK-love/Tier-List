## 中文说明

一个可拖拽的“从夯到拉”排名表生成器。用户可以创建对象（方块），拖到对应档位，并自定义档位。

### 使用方式

1. 打开 `index.html`（双击即可）。
2. 在左侧输入框创建对象，拖拽到右侧档位。
3. 输入档位名称并点击“添加档位”，即可新增档位。
4. 双击对象可改名，右键对象可删除。
5. 点击“删除”可移除某个档位（该档位中的对象会回到待排序区）。
6. 点击“导出图片”可导出当前排名表为 PNG。

### 保存说明

页面会自动保存当前状态（localStorage），重新打开仍会保持上次内容。

## English

A drag-and-drop “Hot to Lame” tier list maker. Create items (blocks), drag them into tiers, and customize tiers.

### Usage

1. Open `index.html` (double-click).
2. Create items in the left panel and drag them into tiers.
3. Enter a tier name and click “Add Tier” to create a new tier.
4. Double-click an item to rename it; right-click to delete it.
5. Click “Delete” to remove a tier (its items return to the pool).
6. Click “Export Image” to download the current tier list as PNG.

### Persistence

The page auto-saves state (localStorage), so reopening will restore the last session.
