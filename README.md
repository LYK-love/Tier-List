# 从夯到拉 Tier List Maker

[中文](README.md) | [English](README.en.md)

![Demo](assets/example.png)

一个纯前端可拖拽的 “从夯到拉” 排名表生成器：创建条目、拖入档位、自由增删/重命名档位，并可一键导出 PNG。  
双击打开 `index.html` 即可使用；状态自动保存到浏览器（localStorage）。
